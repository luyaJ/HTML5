绘制时钟效果

## 知识

`ctx.translate(x,y)`方法:重新设置画布的坐标源点，设置后，x、y坐标变为(x,y)。

`ctx.arc`(圆心x坐标，圆心y坐标，圆半径，起始点弧度，结束点弧度，是否顺时针)绘制圆形或圆弧。

`ctx.save()`和`ctx.restore()`，记录起始状态和回复状态。

## 角度弧度制转换

### 弧度制单位

* 1度 = PI弧度 / 180度
* 1弧度 = 180度 / PI弧度

### 角度转换公式

* 弧度 = 角度*PI/180
* 角度 = 弧度*180/PI
